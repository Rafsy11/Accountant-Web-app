# Brand Guidelines: Accountant Pro

## Visual Identity
- **Vibe**: Professional, Modern, High-Tech, Dark-Mode centric.
- **Theme**: Deep Dark Mode.

## Color Palette
- **Background**: `#0B0E14` (Deep Charcoal Black)
- **Card Background**: `#161B22` (Slightly lighter navy/gray)
- **Primary Accent**: `#00D1FF` (Neon Cyan) - Use for active states and primary buttons.
- **Success**: `#10B981` (Emerald Green)
- **Danger**: `#EF4444` (Rose Red)
- **Neutral/Text**: `#E6EDF3` (Off-white for readability)

## Typography & UI Elements
- **Font**: Inter or System Sans-Serif (Clean and bold).
- **Border Radius**: `16px` (rounded-2xl) for a soft modern look.
- **Shadows**: Subtle glow on primary buttons using `0 0 15px rgba(0, 209, 255, 0.3)`.
- **Glassmorphism**: Use `backdrop-blur-md` on modals and navigation bars.

## Category Color Map (Mandatory)
- **Salary**: `#4BC0C0` (Turquoise)
- **Food**: `#FF6384` (Pinkish Red)
- **Transport**: `#FFCE56` (Yellow)
- **Rent/Utilities**: `#9966FF` (Purple)
- **Entertainment**: `#36A2EB` (Blue)
- **Health**: `#FF9F40` (Orange)
- **Freelance**: `#4BC0C0` (Teal)
